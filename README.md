# SAP Blueprint Project – SHBNM Pvt Ltd

## 📌 Overview
This project presents the blueprint design of a fictitious company **SHBNM Pvt Ltd** using SAP concepts. It demonstrates the integration of **FI, MM, and SD modules** through an Excel-based simulation.

## 🧩 Modules Covered
- **FI (Finance):** Financial transactions  
- **MM (Materials Management):** Procurement & Inventory  
- **SD (Sales & Distribution):** Sales & Billing  

## 🔄 Process Flow
- **Procurement:** PR → PO → Goods Receipt → Invoice  
- **Sales:** Order → Delivery → Billing → Payment  
- **Finance:** Debit & Credit entries  

## 💻 Implementation
The project is implemented in **Microsoft Excel** with sheets for:
- Purchase Process  
- Sales Process  
- Inventory  
- Finance  
- Dashboard  

## ✨ Features
- Status dropdowns  
- Conditional formatting  
- Inventory updates  
- Basic dashboard
